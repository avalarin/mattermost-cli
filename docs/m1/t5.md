## Key decisions

- Layout formula `height - 3` (header + statusbar + input each consume 1 line); viewport fills the rest
- `TestLayoutHeightFitsWindow` validates `viewport.Height == windowHeight - 3` directly on the model struct, avoiding any rendering dependency
