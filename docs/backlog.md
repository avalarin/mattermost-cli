# Backlog — Nice to Have

Фичи и улучшения, отложенные на будущее. Не блокируют текущие вехи.

---

## UI / Рендеринг

- **Markdown в сообщениях** — рендерить `**bold**`, `_italic_`, `` `code` ``, блоки кода через `charmbracelet/glamour`.
- **Emoji unicode** — корректное отображение emoji (зависит от шрифта; добавить fallback на `:emoji_code:`).
- **Встроенный emoji picker** — всплывающий список топ-N emoji с поиском по имени при нажатии `r`.

## Платформы

- **Clipboard на Linux** — поддержка `xclip` / `xsel` / `wl-copy` в дополнение к `pbcopy` (macOS).
- **Clipboard на Windows** — через PowerShell / `clip.exe`.

## AI

- **Суммаризация контекста** — когда история AI-диалога превышает лимит, перед отбрасыванием старых сообщений создавать краткое резюме через отдельный API-вызов.
- **Prompt caching** — кешировать системный промпт + контекст каналов для снижения стоимости повторных вызовов.
- **Счётчик токенов** — показывать расход токенов за сессию в статус-баре AI-панели (`ai.show_token_usage = true`).
- **Флаг `ai.require_confirm`** — отключаемое подтверждение перед действиями AI (сейчас всегда `true`).

## Функциональность

- **Поиск сообщений (M4)** — детальная проработка: FTS через Mattermost REST `/search`, UI панели результатов.
- **Поддержка нескольких серверов** — профили в конфиге, переключение между серверами.
- **Уведомления ОС** — системные нотификации (`osascript` / `notify-send`) при упоминании пользователя.
- **Фильтр в ленте (M1)** — возможность фильтровать общую ленту по каналу без переключения в M2-режим.
- **Группы каналов в боковой панели** — сворачиваемые группы (Channels, Direct Messages, Favorites) аналогично десктопному клиенту Mattermost.
- **Закреплённые (pinned) каналы** — выделенная секция «Pinned» в верхней части списка каналов.
